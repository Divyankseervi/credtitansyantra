# ROI Intelligence Engine

Land ROI analysis with internal scoring and AI (Gemini) investment advice.

## Stack

- **Backend:** Node.js + Express  
- **Frontend:** React (Vite)

## Structure

```
roi-intelligence-engine/
├── backend/
│   ├── app.js          # Express app, POST /analyze-roi
│   ├── roiEngine.js    # Scoring & ROI calculation
│   ├── aiAdvisor.js    # Gemini API advice
│   ├── .env            # PORT, GEMINI_API_KEY
│   └── package.json
├── frontend/
│   ├── src/
│   │   ├── App.jsx     # Form + results UI
│   │   ├── App.css
│   │   ├── main.jsx
│   │   └── index.css
│   ├── index.html
│   ├── vite.config.js
│   └── package.json
└── README.md
```

## Run steps

### 1. Backend

```bash
cd backend
npm install
```

Create/edit `backend/.env`:

```
PORT=5000
GEMINI_API_KEY=your_actual_gemini_api_key
```

Get a key: [Google AI Studio](https://aistudio.google.com/apikey).

Start the API:

```bash
npm start
```

Server runs at **http://localhost:5000**.

### 2. Frontend

In a new terminal:

```bash
cd frontend
npm install
npm run dev
```

App runs at **http://localhost:5173**.

### 3. Use the app

1. Open http://localhost:5173  
2. Enter **lat**, **lon**, **ndvi** (0–1), **change** (improving/stable/declining), **land type**  
3. Click **Analyze**  
4. View **Growth Score**, **Future Value**, **5yr ROI**, **10yr ROI**, and **AI Advice**

## API

**POST** `/analyze-roi`

Request body:

```json
{
  "lat": 22.5,
  "lon": 88.3,
  "ndvi": 0.6,
  "change": "improving",
  "landType": "residential"
}
```

Response:

```json
{
  "growthScore": 42.5,
  "futureValue": 960000,
  "roi5yr": 29.75,
  "roi10yr": 68,
  "aiAdvice": "AI-generated investment advice text..."
}
```

## ROI engine (summary)

- **Urban growth:** NDVI + change (improving/stable/declining)  
- **Infrastructure:** latitude bands  
- **Demand / Livability / Risk:** from change and NDVI  
- **Growth score:** weighted mix, clamped 0–100  
- **Future value:** base 800,000 × multiplier by score band  
- **5yr / 10yr ROI:** derived from growth score  

AI advice is generated by Gemini using location, scores, and land type.
